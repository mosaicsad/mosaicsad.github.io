属性

Setting Description Default
layout Layout post或page
date 创建日期 文件的创建日期
update 修改日期 文件的修改日期
comments 是否开启评论 true
tags 标签
categories 分类

###更新自己的网站

也可简写为

###添加统计功能

####创建Lean Cloud 应用：
LeanCloud平台提供一站式后端云服务，从数据存储、实时聊天、消息推送到移动统计，涵盖应用开发的多方面后端需求。
首先，申请账号。
我们只用到它的数据存储部分,具体步骤如下：

首先到『控制台』创建一个应用，名字随便取。
点击左侧边栏的『存储』选项，选择『创建Class』，取名为”Counter“。
点击新建应用右上角的齿轮，在『应用Key』选项里得到APP ID 和 APP Key，在后面会用到。

####修改_config.yml
添加appid 和 appkey 即可

###设置 Web 安全域名
在你所创建的应用的 设置->安全中心 中设置 Web 安全域名 添加你的域名到 Web 安全域名中（若本地服务也想看到访问量，添加 http://localhost:4000/）
